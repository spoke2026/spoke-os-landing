Line Heights

Line height affects readability and visual spacing. Match to context.

1.0 — Tight
Statistics and large headlines only. Creates visual impact.
1.25 — Compact
Section headers and titles. Balanced spacing.
1.5 — Standard
Body text, descriptions, tables. The default for comfortable reading. Use this for most copy.
1.65 — Open
Prose blocks, long-form content. Generous spacing for extended readable text. Use for articles.

Responsive Design

Design system behavior at different screen sizes.

Breakpoint: 768px

Single primary breakpoint for responsive behavior. Below 768px, layouts adjust as follows:

  • Multi-column grids collapse to 1 column
  • Page padding reduces to 24px horizontal
  • Font sizes may use smaller scale values
  • Spacing gaps reduce proportionally
  • Navigation may become compact or slide-out

Fluid Typography

Use clamp() for responsive font sizes that scale smoothly between breakpoints.

font-size: clamp(24px, 3.5vw, 40px);
Scales from 24px minimum up to 40px maximum based on viewport width.

Flexible Grids

Use grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)) for grids that automatically adjust column count.

Mobile Considerations

  • Touch targets minimum 44x44px
  • Buttons and links should be easily tappable
  • Forms should be single-column on mobile
  • Images should scale responsively
  • Test on actual devices, not just browser resizing
